# Task Management Application

A simple application to track your tasks.


## Features

- Create task with priorty feature
- Update task real time
- Can delete your selected task at ease
- Has 'complete' flag to tag the task as completed
- Sort and Filter is available to ease your checking.


## Documentation

Just do these command after you clone the repository.



```
//redirect to the project after cloning
cd /task-management

// install dependencies
npm install

// to run it locally
npm run dev
```


## Tech Stack

- VueJS (3.5.12)
- Tailwind (3.4.15)
- Vite (5.4.10)

For storing the tasks, I used `localStorage`.
